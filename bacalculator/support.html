Quick Start

  • Open module menu to access TVM, Cash Flow, Bond, Date, Amortization, and Statistics worksheets.
  • Enter values in worksheet fields and trigger compute to solve the unknown variable.
  • Use Settings to adjust decimal places, date format, calculation order (Chn/AOS), and language.

Frequently Asked Questions

Does the app require an internet connection?
No. FinCalc Pro works fully offline for all core financial calculations.
How do I reset worksheet values or memory registers?
Open Settings and use the Memory Panel / Clear Memory actions. Worksheet inputs can also be overwritten directly in each module field.
Why is my result slightly different from another calculator?
Check rounding mode, decimal places, date format, and day-count assumptions. FinCalc Pro uses high-precision Decimal arithmetic and BA-style key flow.
What data is uploaded from my calculations?
None. Calculation data stays on-device. Current release follows the App Store privacy label: Data Not Collected.
