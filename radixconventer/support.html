应用介绍

Radix Converter 是一款优雅、强大的进制转换工具，支持二进制、八进制、十进制、十六进制、Base32、Base36 之间的实时互相转换。

实时转换

输入即转换，无需点击按钮

精美设计

支持深色模式，每个进制独特配色

多语言

支持中文、英语、日语等6种语言

常见问题

如何使用？

1. 在输入框中输入数字
2. 选择当前进制（二进制、八进制、十进制等）
3. 应用会自动显示其他所有进制的转换结果
4. 点击任意结果即可复制到剪贴板

支持哪些进制？

  • 二进制 (Binary) - Base 2，使用 0-1
  • 八进制 (Octal) - Base 8，使用 0-7
  • 十进制 (Decimal) - Base 10，使用 0-9
  • 十六进制 (Hexadecimal) - Base 16，使用 0-9, A-F
  • 32进制 (Base32) - Base 32，使用 0-9, A-V
  • 36进制 (Base36) - Base 36，使用 0-9, A-Z

如何复制结果？

点击任意转换结果，数值会自动复制到剪贴板。您可以在设置中选择复制时是否包含进制前缀（如 0x、0b）。

如何切换语言？

点击右上角设置图标，选择"语言"，即可切换到您偏好的语言。应用支持中文、英语、日语、葡萄牙语、西班牙语、俄语。

如何切换深色/浅色模式？

点击右上角设置图标，选择"主题"，可以选择自动（跟随系统）、浅色或深色模式。

应用会收集我的数据吗？

不会。Radix Converter 完全在您的设备上本地运行，不收集任何个人信息，不需要网络连接。详见隐私政策。

使用技巧

  • 切换进制时，应用会自动将当前输入值转换到新进制
  • 输入无效字符时，输入框会显示红色边框提示
  • 长按清除按钮可以快速重置所有内容
  • 每个进制都有独特的渐变配色，方便识别
