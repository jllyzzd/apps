Política de Privacidad

Fecha de Vigencia: 25 de diciembre de 2025

Bienvenido a SatsClock ("la App"). Valoramos su privacidad y estamos comprometidos a proteger su información personal. Esta Política de Privacidad explica nuestras prácticas con respecto a su información.

1. No Recopilamos Datos

SatsClock es un panel de Bitcoin con prioridad de privacidad. No recopilamos, almacenamos, transmitimos ni compartimos ninguna de sus informaciones personales, datos de uso o información del dispositivo.

  • Sin Cuentas: No necesita registrarse ni iniciar sesión para usar ninguna función de la App.
  • Sin Rastreadores: No utilizamos ningún SDK de análisis de terceros ni servicios de seguimiento del comportamiento del usuario.
  • Sin Anuncios: La App no contiene publicidad ni muestra anuncios de redes de terceros.
  • Procesamiento Local: Toda la visualización de datos y cálculos se realizan localmente en su dispositivo.

2. Solicitudes de Red

La App obtiene datos de blockchain de Bitcoin públicos de las siguientes API de terceros:

  • API de Mempool.space: Datos de blockchain en tiempo real (altura de bloque, tarifas, progreso de halving)
  • API de Binance: Precio de mercado actual de Bitcoin

Importante: Estas solicitudes de API son anónimas. No se transmiten identificadores personales, información del dispositivo o patrones de uso. La App solo lee datos de blockchain públicos.

3. Almacenamiento de Datos

La App almacena datos mínimos de preferencias del usuario en su dispositivo únicamente para mejorar su experiencia:

  • Selección de Tema: Su esquema de color y configuraciones de efectos preferidas
  • Preferencia de Idioma: Idioma de interfaz seleccionado (8 idiomas compatibles)
  • Configuraciones de Visualización: Intervalo de actualización, pantalla encendida, configuraciones de visualización de citas
  • Datos en Caché: Datos de blockchain en caché temporal para reducir solicitudes de red

Todos los datos se almacenan localmente a través de iOS UserDefaults y nunca se transmiten a ningún servidor.

4. App Groups y Uso Compartido de Datos

La App utiliza iOS App Groups para permitir la funcionalidad de widgets:

  • Propósito: Compartir datos de blockchain con widgets de iOS y complicaciones de Apple Watch
  • Almacenamiento: Los datos se almacenan en el contenedor UserDefaults compartido en su dispositivo
  • Sin Sincronización en la Nube: Los datos nunca se suben a ningún servicio en la nube o servidor de terceros

5. Conectividad de la App de Watch

Si utiliza la app de Apple Watch, se realiza el siguiente uso compartido de datos a través del framework WatchConnectivity de Apple:

  • iPhone → Watch: Precio de Bitcoin, altura de bloque, tarifas, configuraciones de tema y cadenas localizadas
  • Almacenamiento: Todos los datos se almacenan localmente en su dispositivo Watch
  • Privacidad: No se transmiten datos a ningún servidor externo. Toda la comunicación permanece en sus dispositivos locales.

6. Permisos

La App no requiere ningún permiso especial del sistema (como Cámara, Ubicación, Micrófono, Contactos, Fotos, etc.) para funcionar. La App solo utiliza:

  • Acceso a Internet: Para obtener datos de blockchain pública (solicitudes anónimas)
  • Notificaciones Locales: Opcionales (solo si habilita las actualizaciones de línea de tiempo del widget)

7. Privacidad Infantil

La App es adecuada para usuarios de todas las edades. Dado que no recopilamos ninguna información personal, no hay recopilación o procesamiento de datos de privacidad infantil.

8. Fuente de Datos

Los datos de Bitcoin mostrados en la App se obtienen de:

  • Mempool.space: Explorador de blockchain de Bitcoin de código abierto (https://mempool.space)
  • Binance: API de exchange de criptomonedas para datos de precios

No respaldamos ni garantizamos la precisión de los datos de estas fuentes de terceros.

9. Cambios en Esta Política

Aunque es improbable un cambio significativo dado nuestro diseño con prioridad de privacidad, podemos actualizar esta Política de Privacidad de vez en cuando. Cualquier cambio se publicará en esta página con una fecha de vigencia actualizada.

10. Sus Derechos

Dado que no recopilamos datos, no hay datos personales para que usted solicite acceder, modificar o eliminar. Todos los datos almacenados en su dispositivo se pueden borrar mediante:

  • Desinstalar la App (borra todos los datos locales)
  • Ir en la App a Configuración → Restablecer Configuración
