Politique de Confidentialité

Date d'Entrée en Vigueur : 25 décembre 2025

Bienvenue dans SatsClock ("l'Application"). Nous valorisons votre vie privée et nous sommes engagés à protéger vos informations personnelles. Cette Politique de Confidentialité explique nos pratiques concernant vos informations.

1. Nous Ne Collectons Aucune Donnée

SatsClock est un tableau de Bitcoin privilégiant la confidentialité. Nous ne collectons, ne stockons, ne transmettons ni ne partageons aucune de vos informations personnelles, données d'utilisation ou informations sur l'appareil.

  • Pas de Comptes : Vous n'avez pas besoin de vous inscrire ou de vous connecter pour utiliser les fonctionnalités de l'Application.
  • Pas de Traceurs : Nous n'utilisons aucun SDK d'analyse tiers ou services de suivi du comportement des utilisateurs.
  • Pas de Publicité : L'Application ne contient pas de publicité et n'affiche pas d'annonces de réseaux tiers.
  • Traitement Local : Toutes les affichages de données et calculs sont effectués localement sur votre appareil.

2. Requêtes Réseau

L'Application récupère des données de blockchain Bitcoin publiques auprès des API tiers suivantes :

  • API Mempool.space : Données de blockchain en temps réel (hauteur de bloc, frais, progression du halving)
  • API Binance : Prix du marché actuel du Bitcoin

Important : Ces requêtes API sont anonymes. Aucun identifiant personnel, information sur l'appareil ou modèle d'utilisation n'est transmis. L'Application lit uniquement les données de blockchain publiques.

3. Stockage des Données

L'Application stocke des données minimales de préférences utilisateur sur votre appareil uniquement pour améliorer votre expérience utilisateur :

  • Sélection de Thème : Votre schéma de couleurs et paramètres d'effets préférés
  • Préférence Linguistique : Langue de l'interface sélectionnée (8 langues prises en charge)
  • Paramètres d'Affichage : Intervalle d'actualisation, maintien de l'écran allumé, paramètres d'affichage des citations
  • Données en Cache : Données de blockchain en cache temporaire pour réduire les requêtes réseau

Toutes les données sont stockées localement via iOS UserDefaults et ne sont jamais transmises à un serveur.

4. App Groups et Partage de Données

L'Application utilise iOS App Groups pour permettre la fonctionnalité des widgets :

  • Objet : Partager des données de blockchain avec les widgets iOS et les complications Apple Watch
  • Stockage : Les données sont stockées dans le conteneur UserDefaults partagé sur votre appareil
  • Pas de Synchronisation Cloud : Les données ne sont jamais téléchargées vers un service cloud ou un serveur tiers

5. Connectivité de l'App Watch

Si vous utilisez l'application Apple Watch, le partage de données suivant s'effectue via le framework WatchConnectivity d'Apple :

  • iPhone → Watch : Prix du Bitcoin, hauteur de bloc, frais, paramètres de thème et chaînes localisées
  • Stockage : Toutes les données sont stockées localement sur votre appareil Watch
  • Confidentialité : Aucune donnée n'est transmise à un serveur externe. Toute la communication reste sur vos appareils locaux.

6. Autorisations

L'Application ne nécessite aucune autorisation système spéciale (comme Appareil Photo, Localisation, Microphone, Contacts, Photos, etc.) pour fonctionner. L'Application utilise uniquement :

  • Accès Internet : Pour récupérer des données de blockchain publiques (requêtes anonymes)
  • Notifications Locales : Optionnelles (seulement si vous activez les mises à jour de la timeline du widget)

7. Confidentialité des Enfants

L'Application convient aux utilisateurs de tous âges. Comme nous ne collectons aucune information personnelle, il n'y a pas de collecte ou de traitement de données sur la confidentialité des enfants.

8. Source des Données

Les données Bitcoin affichées dans l'Application sont obtenues de :

  • Mempool.space : Explorateur de blockchain Bitcoin open source (https://mempool.space)
  • Binance : API d'échange de cryptomonnaie pour les données de prix

Nous n'approuvons ne garantissons pas l'exactitude des données de ces sources tierces.

9. Modifications de Cette Politique

Bien que des changements significatifs soient improbables compte tenu de notre conception privilégiant la confidentialité, nous pouvons mettre à jour cette Politique de Confidentialité de temps à autre. Tout changement sera publié sur cette page avec une date d'entrée en vigueur mise à jour.

10. Vos Droits

Comme nous ne collectons pas de données, il n'y a pas de données personnelles pour vous dont vous pouvez demander l'accès, la modification ou la suppression. Toutes les données stockées sur votre appareil peuvent être effacées en :

  • Désinstallant l'Application (efface toutes les données locales)
  • Allant dans l'Application dans Paramètres → Réinitialiser les Paramètres
