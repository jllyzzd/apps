SatsClock 소개

SatsClock은 iOS 시스템과 깊이 통합된 실시간 비트코인 대시보드 앱입니다. 아름다운 SF 테마 인터페이스로 가격, 블록 높이, 메모풀 수수료, 반감기 카운트다운, 모스크바 시간 등 실시간 블록체인 데이터를 표시합니다.

주요 기능

📊 실시간 데이터 표시

  • 비트코인 가격: USD의 실시간 가격 및 7일 추세 차트
  • 블록 높이: 현재 블록체인 블록 번호(형식화됨)
  • 메모풀 수수료: 최빠름/30분/경제 수수료율(sats/vB)
  • 모스크바 시간: 1 USD로 살 수 있는 sats 수(1 USD ÷ BTC 가격)
  • 반감기 진행 상황: 다음 비트코인 반감기까지 남은 일/블록 수

🎨 6가지 프로 테마

비트코인 오렌지 - 클래식 전문
매트릭스 그린 - 사이버펑크 해커
순수 흰색 - 미니멀리스트 우아함
픽셀 레트로 - 8비트 향수
네온 테크 - 미래지향적 사이버펑크
터미널 모노 - 개발자 스타일

각 테마에는 사용자 정의 글꼴, 색상 구성표, 독립적으로 전환할 수 있는 시각 효과가 포함되어 있습니다.

🔌 깊은 iOS 통합

  • StandBy 모드: 충전 중인 iPhone에 최적화된 가로 레이아웃
  • 홈 화면 위젯: 소형, 중형, 대형의 3가지 크기
  • 잠금 화면 위젯: 원형, 직사각형, 인라인
  • Apple Watch 앱: 기능이 완비한 watchOS 앱 및 컴플리케이션
  • Spotlight 검색: 비트코인 데이터에 대한 빠른 액세스

자주 묻는 질문(FAQ)

1. 설정을 여는 방법은 무엇인가요?

화면의 아무 곳이나 0.6초 동안 길게 누르면 설정이 열립니다. 또는 인터페이스 선택기(하단 버튼) → 인터페이스 선택 → 우측 상단의 기어 아이콘 탭을 통해 설정에 액세스할 수도 있습니다.

2. "모스크바 시간"이란 무엇인가요?

모스크바 시간은 다음과 같이 계산되는 비트코인 지표입니다: 100,000,000 sats ÷ BTC USD 가격. 이는 1달러로 살 수 있는 사토시(sats) 수를 나타냅니다. 가격이 오르면 모스크바 시간은 내려가고 그 반대의 경우도 마찬가지입니다.

예: BTC가 100,000달러인 경우 모스크바 시간 = 1,000 sats/USD입니다.

3. StandBy 모드를 활성화하는 방법은 무엇인가요?

StandBy 모드는 iOS 17+를 실행하는 iPhone 15 Pro 이상에서 자동으로 활성화됩니다. iPhone을가로 방향으로 MagSafe 또는 Qi 무선 충전기에 올려두기만 하면 됩니다. 앱이 자동으로 StandBy 레이아웃으로 전환됩니다.

4. 데이터는 얼마나 자주 새로 고쳐지나요?

기본적으로 데이터는 15초마다 새로 고쳐집니다. 설정(길게 누름 → 데이터 → 새로 고침 간격)에서 10~60초로 조정할 수 있습니다. 참고: 더 자주 새로 고치면 배터리 수명에 약간 영향을 줄 수 있습니다.

5. 데이터가 업데이트되지 않는 이유는 무엇인가요?

다음을 확인하십시오:

  • 활성 인터넷 연결(Wi-Fi 또는 셀룰러)이 있는지 확인합니다
  • 화면 하단의 "마지막 업데이트" 타임스탬프를 확인합니다
  • Wi-Fi를 사용하는 경우 셀룰러 데이터로 전환해 보십시오
  • 네트워크 문제로 인해 일시적인 지연이 발생할 수 있습니다 - 앱이 자동으로 다시 시도합니다

6. 홈 화면에 위젯을 추가하는 방법은 무엇인가요?

  1. 홈 화면의 빈 공간을 길게 눌러 앱이 흔들릴 때까지 기다립니다
  2. 좌측 상단의 + 버튼을 탭합니다
  3. "SatsClock"을 검색하거나 스크롤하여 찾습니다
  4. 원하는 위젯 크기를 선택합니다
  5. 위젯 추가를 탭합니다

7. 워치 페이스를 사용자 지정하는 방법은 무엇인가요?

  1. 현재 워치 페이스를 길게 누릅니다
  2. 오른쪽으로 스와이프하여 새로 만들기를 탭합니다
  3. 컴플리케이션을 지원하는 페이스(모듈식, 크로노그래프 등)를 선택합니다
  4. 컴플리케이션 슬롯을 탭하고 SatsClock을 선택합니다

8. 7가지 다른 보기 모드는 무엇인가요?

하단 인터페이스 버튼을 탭하여 전환합니다:

  • 대시보드: 모든 비트코인 메트릭 개요
  • 가격 세부정보: 가격 및 7일 추세 차트
  • 블록 세부정보: 블록 높이 및 사이클 진행 상황
  • 반감기 세부정보: 완전한 반감기 진행 상황 시각화
  • 모스크바 시간: 전용 블록 클럭 보기
  • 반감기 카운트다운: 다음 반감기까지의 일 및 블록 수
  • 수수료율: 네트워크 수수료율 비교

9. 배터리를 절약하기 위해 시각 효과를 비활성화하는 방법은 무엇인가요?

가능합니다. 설정(길게 누름) → 표시 → 성능 모드로 이동합니다. 선택한 테마의 색상과 글꼴을 유지하면서 모든 텍스트 광 효과 및 배경 애니메이션을 비활성화합니다.

10. 앱이 오프라인으로 작동하나요?

예, 제한이 있습니다. 앱은 오프라인일 때 마지막 캐시된 데이터를 표시합니다. 그러나 실시간 업데이트를 가져오려면 인터넷 연결이 필요합니다. 위젯 및 워치 컴플리케이션도 캐시된 데이터를 표시합니다.

11. 반감기 날짜는 어떻게 계산되나요?

반감기 날짜는 다음을 기준으로 추정됩니다:

  • 현재 블록 높이(mempool.space에서 가져옴)
  • 다음 210,000 블록 마일스톤까지 남은 블록 수
  • 평균 10분 블록 시간

이것은 추정치입니다. 네트워크 해시레이트 변경에 따라 실제 반감기 시간은 다를 수 있습니다.

12. 앱이 인터넷 액세스를 필요로 하는 이유는 무엇인가요?

앱은 다음에서 공개 블록체인 데이터를 가져오기 위해 인터넷 액세스가 필요합니다:

  • Mempool.space API: 블록 높이, 수수료율, 반감기 진행 상황
  • Binance API: 현재 비트코인 시장 가격

모든 요청은 익명이며 개인 정보가 포함되어 있지 않습니다.

13. 화면 켜기 유지를 활성화하는 방법은 무엇인가요?

설정(길게 누름) → 표시 → 화면 켜기 유지로 이동합니다. 이렇게 하면 앱이 열려 있는 동안 iPhone이 절전 모드로 들어가지 않습니다. 전용 모니터링 설정이나 StandBy 모드에 유용합니다.

14. 인용문 캐러셀 기능이란 무엇인가요?

앱은 사토시 나카모토, 핼 피니 등 비트코인 유명인사의 인용문을 순환하여 표시합니다. 설정(길게 누름) → 표시 → 인용문 표시에서 비활성화할 수 있습니다.

팁 및 모범 사례

  • StandBy 모드: MagSafe 충전기와 함께 사용하면 최상의 항상 켜기 디스플레이 경험을 얻을 수 있습니다
  • 저전력 모드: iOS는 저전력 모드에서 시각 효과를 자동으로 비활성화합니다

  • 워치 페이스: 최상의 컴플리케이션 레이아웃을 위해 모듈식 또는 인포그래프 페이스를 사용합니다

문제 해결

위젯이 업데이트되지 않음

위젯은 iOS 일정에 따라 업데이트되며 지속적이지 않습니다. 위젯을 강제로 업데이트하려면:

  1. SatsClock 메인 앱을 엽니다(새 데이터를 가져옴)
  2. iOS가 위젯을 업데이트할 때까지 15-30초 기다립니다
  3. 위젯을 길게 누르고 "위젯 새로 고침"을 탭할 수도 있습니다

워치 앱에 "데이터 없음"이 표시됨

Apple Watch 앱에 데이터가 표시되지 않는 경우:

  1. iPhone이 근처에 있고 블루투스가 활성화되어 있는지 확인합니다
  2. iPhone에서 SatsClock을 열어 데이터를 새로 고칩니다
  3. 워치 앱은 1-2분 내에 동기화되어야 합니다
  4. 문제가 지속되면 워치 앱의 페어링을 해제하고 다시 페어링합니다

앱 시작 시 충돌

앱이 충돌하는 경우:

  • iOS 17.0 이상을 실행하는지 확인합니다
  • App Store에서 앱을 삭제하고 다시 설치합니다(사용자 데이터를 저장하지 않으므로 데이터는 손실되지 않습니다)
  • iPhone을 다시 시작합니다
