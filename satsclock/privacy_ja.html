プライバシーポリシー

発効日：2025年12月25日

SatsClock（「本アプリ」）へようこそ。私たちはお客様のプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、情報に関する当社の practices について説明します。

1. データは収集しません

SatsClock はプライバシー優先のビットコインダッシュボードアプリケーションです。私たちはお客様の個人情報、使用データ、またはデバイス情報を収集、保存、送信、または共有しません。

  • アカウント不要： アプリの機能を使用するために登録またはログインする必要はありません。
  • トラッカーなし： 第三者の分析 SDK やユーザー行動追跡サービスは使用していません。
  • 広告なし： 本アプリには広告は含まれておらず、第三者ネットワークから広告を配信することもありません。
  • ローカル処理： すべてのデータ表示と計算はデバイス上でローカルに実行されます。

2. ネットワークリクエスト

本アプリは、以下の第三者 API から公開ビットコインブロックチェーンデータを取得します：

  • Mempool.space API： リアルタイムブロックチェーンデータ（ブロック高さ、手数料、半減期進捗）
  • Binance API： 現在のビットコイン市場価格

重要： これらの API リクエストは匿名です。個人識別子、デバイス情報、または使用パターンは送信されません。アプリは公開ブロックチェーンデータのみを読み取ります。

3. データ保存

本アプリは、ユーザーエクスペリエンスを向上させるためにのみ、デバイス上に最小限のユーザー設定データを保存します：

  • テーマ選択： 選択したカラーテーマと効果設定
  • 言語設定： 選択したインターフェース言語（8言語対応）
  • 表示設定： 更新間隔、画面オン設定、引用表示設定
  • キャッシュデータ： ネットワークリクエストを減らすための一時的なブロックチェーンデータキャッシュ

すべてのデータは iOS UserDefaults を通じてローカルに保存され、サーバーに送信されることはありません。

4. App Groups とデータ共有

本アプリは iOS App Groups を使用してウィジェット機能を実現しています：

  • 目的： iOS ウィジェットおよび Apple Watch コプリケーションとのブロックチェーンデータ共有
  • 保存： デバイス上の共有 UserDefaults コンテナに保存されます
  • クラウド同期なし： データはクラウドサービスまたは第三者サーバーにアップロードされません

5. ウォッチアプリ接続

Apple Watch アプリを使用する場合、Apple の WatchConnectivity フレームワークを通じて以下のデータ共有が行われます：

  • iPhone → Watch： ビットコイン価格、ブロック高さ、手数料、テーマ設定、ローカライズ文字列
  • 保存： すべてのデータはウォッチデバイス上にローカルに保存されます
  • プライバシー： 外部サーバーへのデータ送信はありません。すべての通信はローカルデバイス内で行われます。

6. 権限

本アプリは動作時に特別なシステム権限（カメラ、位置情報、マイク、連絡先、写真など）を必要としません。本アプリが使用するのは以下のみです：

  • インターネットアクセス： 公開ブロックチェーンデータの取得（匿名リクエスト）
  • ローカル通知： オプション（ウィジェットタイムライン更新を有効にした場合のみ）

7. 子供のプライバシー

本アプリはすべての年齢層のユーザーに適しています。個人情報を収集しないため、子供のプライバシーデータの収集や処理は行いません。

8. データソース

本アプリに表示されるビットコインデータは以下から取得されます：

  • Mempool.space： オープンソースビットコインブロックチェーンエクスプローラー（https://mempool.space）
  • Binance： 価格データの暗号通貨取引所 API

これらの第三者ソースからのデータの正確性を保証または支持するものではありません。

9. ポリシーの変更

プライバシー優先の設計のため、重大な変更は可能性が低いですが、本プライバシーポリシーを随時更新する場合があります。変更はこのページに更新日とともに投稿されます。

10. あなたの権利

データを収集しないため、アクセス、変更、削除を要求する個人データはありません。デバイス上の保存データは以下でクリアできます：

  • アプリをアンインストールする（すべてのローカルデータをクリア）
  • アプリ内で設定 → 設定をリセット
