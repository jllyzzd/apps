隐私政策

生效日期：2025年12月25日

欢迎使用 SatsClock（"本应用"）。我们重视您的隐私，并致力于保护您的个人信息。本隐私政策说明了我们关于您信息的使用实践。

1. 我们不收集任何数据

SatsClock 是一款隐私优先的比特币仪表盘应用。我们不收集、存储、传输或共享您的任何个人信息、使用数据或设备信息。

  • 无需账号： 您无需注册或登录即可使用应用的任何功能。
  • 无追踪器： 我们不使用任何第三方分析 SDK 或用户行为追踪服务。
  • 无广告： 本应用不含广告，也不展示任何第三方网络广告。
  • 本地处理： 所有数据显示和计算均在您的设备本地执行。

2. 网络请求

本应用从以下第三方 API 获取公共比特币区块链数据：

  • Mempool.space API： 实时区块链数据（区块高度、手续费、减半进度）
  • Binance API： 当前比特币市场价格

重要提示： 这些 API 请求是匿名的。不会传输任何个人标识符、设备信息或使用模式。应用仅读取公共区块链数据。

3. 数据存储

本应用在您的设备上存储最少的用户偏好数据，仅用于提升用户体验：

  • 主题选择： 您偏好的颜色主题和效果设置
  • 语言偏好： 选定的界面语言（支持8种语言）
  • 显示设置： 刷新间隔、保持屏幕常亮、语录显示设置
  • 缓存数据： 临时缓存的区块链数据以减少网络请求

所有数据通过 iOS UserDefaults 本地存储，绝不传输到任何服务器。

4. App Groups 和数据共享

本应用使用 iOS App Groups 实现小组件功能：

  • 用途： 与 iOS 小组件和 Apple Watch 复杂功能共享区块链数据
  • 存储： 数据存储在您设备的共享 UserDefaults 容器中
  • 无云同步： 数据绝不会上传到任何云服务或第三方服务器

5. 手表应用连接

如果您使用 Apple Watch 应用，将通过 Apple 的 WatchConnectivity 框架进行以下数据共享：

  • iPhone → Watch： 比特币价格、区块高度、费率、主题设置和本地化字符串
  • 存储： 所有数据存储在您的 Watch 设备本地
  • 隐私： 不会向任何外部服务器传输数据。所有通信均保留在您的本地设备上。

6. 权限

本应用不需要任何特殊系统权限（如相机、位置、麦克风、通讯录、照片等）即可运行。本应用仅使用：

  • 互联网访问： 获取公共区块链数据（匿名请求）
  • 本地通知： 可选（仅当您启用小组件时间线更新时）

7. 儿童隐私

本应用适合所有年龄段用户。由于我们不收集任何个人信息，因此不会收集或处理儿童隐私数据。

8. 数据来源

本应用显示的比特币数据来源于：

  • Mempool.space： 开源比特币区块链浏览器（https://mempool.space）
  • Binance： 用于价格数据的加密货币交易所 API

我们不认可或保证这些第三方来源数据的准确性。

9. 本政策变更

尽管考虑到我们的隐私优先设计，重大变更的可能性不大，但我们可能会不时更新本隐私政策。任何变更将在此页面发布并更新生效日期。

10. 您的权利

由于我们不收集数据，因此没有个人数据供您请求访问、修改或删除。存储在您设备上的所有数据可通过以下方式清除：

  • 卸载应用（清除所有本地数据）
  • 在应用内进入设置 → 重置设置
