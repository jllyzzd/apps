À Propos de SatsClock

SatsClock est un tableau de bord Bitcoin en temps réel avec une intégration profonde du système iOS. Il affiche des données de blockchain en direct, y compris le prix, la hauteur de bloc, les frais de mempool, le compte à rebours du halving et Moscow Time, le tout dans une interface thématique de science-fiction impressionnante.

Fonctionnalités Principales

📊 Affichage de Données en Temps Réel

  • Prix Bitcoin : Prix en direct en USD avec graphique de tendance sur 7 jours
  • Hauteur de Bloc : Numéro de bloc actuel de la blockchain avec formatage
  • Frais Mempool : Taux de frais Plus Rapide/Demi Heure/Économique en sats/vB
  • Moscow Time : Combien de sats un USD peut acheter (1 USD ÷ Prix BTC)
  • Progression du Halving : Jours/blocs restants jusqu'au prochain halving Bitcoin

Foire Aux Questions (FAQ)

1. Comment ouvrir les Paramètres ?

Appuyez longuement n'importe où sur l'écran pendant 0,6 seconde pour ouvrir les Paramètres. Alternativement, vous pouvez accéder aux Paramètres via le sélecteur d'interface (bouton inférieur) → sélectionner une interface → toucher l'icône d'engrenage dans le coin supérieur droit.

2. Qu'est-ce que "Moscow Time" ?

Moscow Time est un indicateur Bitcoin calculé comme : 100 000 000 sats ÷ Prix BTC en USD. Il représente combien de satoshis (sats) un dollar américain peut acheter. Lorsque le prix monte, Moscow Time descend, et vice versa.

Exemple : Si BTC est 100 000 $, alors Moscow Time = 1 000 sats/USD.

3. Comment activer le mode StandBy ?

Le mode StandBy est automatique sur iPhone 15 Pro et plus récent exécutant iOS 17+. Placez simplement votre iPhone en orientation horizontale sur un chargeur sans fil MagSafe ou Qi. L'Application passera automatiquement à la mise en page StandBy.

4. À quelle fréquence les données sont-elles actualisées ?

Par défaut, les données sont actualisées toutes les 15 secondes. Vous pouvez ajuster cela dans les Paramètres (appui long → Données → Intervalle d'Actualisation) de 10 à 60 secondes. Remarque : Une actualisation plus fréquente peut affecter légèrement l'autonomie de la batterie.

5. Pourquoi mes données ne sont-elles pas actualisées ?

Veuillez vérifier ce qui suit :

  • Assurez-vous d'avoir une connexion internet active (Wi-Fi ou cellulaire)
  • Vérifiez l'horodatage "Dernière Actualisation" en bas de l'écran
  • Si vous utilisez Wi-Fi, essayez de passer aux données cellulaires
  • Des problèmes réseau peuvent causer des retards temporaires - l'Application réessaiera automatiquement

6. Comment ajouter des widgets à l'écran d'accueil ?

  1. Appuyez longuement sur une zone vide de votre écran d'accueil jusqu'à ce que les applications tremblent
  2. Touchez le bouton + dans le coin supérieur gauche
  3. Recherchez "SatsClock" ou faites défiler pour le trouver
  4. Sélectionnez la taille de widget souhaitée
  5. Touchez Ajouter un Widget

7. Comment personnaliser mon cadran de montre ?

  1. Appuyez longuement sur votre cadran actuel
  2. Faites glisser tout à fait à droite et touchez Nouveau
  3. Sélectionnez un cadran prenant en charge les complications (modulaire, chronographe, etc.)
  4. Touchez un emplacement de complication et sélectionnez SatsClock

8. Quels sont les 7 modes d'affichage différents ?

Touchez le bouton d'interface inférieur pour basculer entre :

  • Tableau de Bord : Aperçu de toutes les métriques Bitcoin
  • Détails du Prix : Prix avec graphique de tendance sur 7 jours
  • Détails du Bloc : Hauteur de bloc et progression du cycle
  • Détails du Halving : Visualisation complète de la progression du halving
  • Moscow Time : Vue d'horloge de bloc dédiée
  • Compte à Rebours du Halving : Jours et blocs jusqu'au prochain halving
  • Taux de Frais : Comparaison des taux de frais du réseau

9. Comment désactiver les effets visuels pour économiser la batterie ?

Oui. Allez dans Paramètres (appui long) → Affichage → Mode Performance. Cela désactive tous les effets de lueur de texte et animations d'arrière-plan tout en conservant les couleurs et polices de votre thème choisi.

10. L'application fonctionne-t-elle hors ligne ?

Oui, avec des limites. L'Application affichera les dernières données mises en cache lorsqu'elle est hors ligne. Cependant, pour obtenir des mises à jour en temps réel, une connexion internet est requise. Les widgets et les complications de Watch afficheront également les données mises en cache.

11. Comment la date du halving est-elle calculée ?

La date du halving est estimée sur la base de :

  • Hauteur de bloc actuelle (récupérée de mempool.space)
  • Blocs restants jusqu'au prochain jalon de 210 000 blocs
  • Temps de bloc moyen de 10 minutes

Il s'agit d'une estimation. Le temps réel du halving peut varier en fonction des changements du taux de hachage du réseau.

12. Pourquoi l'application a-t-elle besoin d'un accès internet ?

L'Application a besoin d'un accès internet pour récupérer des données de blockchain publiques de :

  • API Mempool.space : Hauteur de bloc, taux de frais, progression du halving
  • API Binance : Prix du marché actuel du Bitcoin

Toutes les requêtes sont anonymes et ne contiennent pas d'informations personnelles.

13. Comment activer le maintien de l'écran allumé ?

Allez dans Paramètres (appui long) → Affichage → Maintenir l'Écran Allumé. Cela empêchera votre iPhone de se mettre en veille pendant que l'Application est ouverte. Utile pour les configurations de surveillance dédiées ou le mode StandBy.

14. Qu'est-ce que la fonction de carrousel de citations ?

L'Application affiche des citations rotatives de figures du Bitcoin comme Satoshi Nakamoto, Hal Finney et d'autres. Vous pouvez désactiver cela dans Paramètres (appui long) → Affichage → Afficher les Citations.

Conseils et Meilleures Pratiques

  • Mode StandBy : Utilisez avec un chargeur MagSafe pour la meilleure expérience d'écran toujours allumé
  • Mode Économie d'Énergie : iOS désactive automatiquement les effets visuels en mode économie d'énergie
  • Placement des Widgets : Ajoutez plusieurs widgets pour différentes métriques (prix, bloc, Moscow Time)
  • Cadran de Montre : Utilisez des cadrans Modulaires ou Infograph pour la meilleure disposition des complications
  • Performance : Désactivez les effets sur les modèles d'iPhone plus anciens pour des performances plus fluides

Limitations Connues

  • Apple Watch : L'application Watch nécessite que l'application compagnon iPhone soit exécutée au moins une fois pour synchroniser les données via App Groups
  • Mises à jour en Arrière-plan : Les widgets sont mis à jour selon le programme d'iOS (généralement toutes les 15-30 minutes), pas instantanément
  • Données de Prix : Le prix est affiché uniquement en USD (le support multi-devises est prévu pour les versions futures)

Dépannage

Les widgets ne se mettent pas à jour

Les widgets sont mis à jour selon le programme d'iOS, pas en continu. Pour forcer une mise à jour des widgets :

  1. Ouvrez l'application principale SatsClock (cela récupère des données fraîches)
  2. Attendez 15-30 secondes pour qu'iOS mette à jour les widgets
  3. Vous pouvez également appuyer longuement sur le widget et toucher "Actualiser le Widget"

L'application Watch affiche "Aucune Donnée"

Si votre application Apple Watch n'affiche pas de données :

  1. Assurez-vous que votre iPhone est à proximité et que Bluetooth est activé
  2. Ouvrez SatsClock sur votre iPhone pour actualiser les données
  3. L'application Watch devrait se synchroniser dans les 1-2 minutes
  4. Si les problèmes persistent, dissociez et réassociez l'application Watch

L'application plante au démarrage

Si l'application plante :

  • Assurez-vous d'exécuter iOS 17.0 ou ultérieur
  • Supprimez et réinstallez l'application depuis l'App Store (vous ne perdrez aucune donnée car nous ne stockons pas de données utilisateur)
  • Redémarrez votre iPhone
