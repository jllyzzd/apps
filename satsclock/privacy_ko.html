개인정보 처리방침

시행일: 2025년 12월 25일

SatsClock("본 앱")에 오신 것을 환영합니다. 당사는 귀하의 개인정보를 소중히 여기며 보호하기 위해 최선을 다합니다. 본 개인정보 처리방침은 귀하의 정보에 관한 당사의 관행을 설명합니다.

1. 데이터를 수집하지 않습니다

SatsClock은 개인정보 우선 비트코인 대시보드입니다. 당사는 귀하의 개인정보, 사용 데이터 또는 장치 정보를수집, 저장, 전송 또는 공유하지 않습니다.

  • 계정 불필요: 앱의 모든 기능을 사용하기 위해 가입하거나 로그인할 필요가 없습니다.
  • 추적기 없음: 제3자 분석 SDK나 사용자 행동 추적 서비스를 사용하지 않습니다.
  • 광고 없음: 본 앱에는 광고가 포함되어 있지 않으며 제3자 네트워크에서 광고를 게하지도 않습니다.
  • 로컬 처리: 모든 데이터 표시 및 계산은 장치에서 로컬로 수행됩니다.

2. 네트워크 요청

본 앱은 다음 제3자 API에서 공개 비트코인 블록체인 데이터를 가져옵니다:

  • Mempool.space API: 실시간 블록체인 데이터(블록 높이, 수수료, 반감기 진행 상황)
  • Binance API: 현재 비트코인 시장 가격

중요: 이러한 API 요청은 익명입니다. 개인 식별자, 장치 정보 또는 사용 패턴이 전송되지 않습니다. 앱은 공개 블록체인 데이터만 읽습니다.

3. 데이터 저장

본 앱은 사용자 경험을 향상시키기 위해 장치에 최소한의 사용자 기본 설정 데이터를 저장합니다:

  • 테마 선택: 선호하는 색상 테마 및 효과 설정
  • 언어 기본 설정: 선택한 인터페이스 언어(8개 언어 지원)
  • 표시 설정: 새로 고침 간격, 화면 켜기 기본 설정, 인용문 표시 설정
  • 캐시 데이터: 네트워크 요청을 줄이기 위한 임시 블록체인 데이터 캐시

모든 데이터는 iOS UserDefaults를 통해 로컬로 저장되며 어떤 서버로도 전송되지 않습니다.

4. App Groups 및 데이터 공유

본 앱은 iOS App Groups를 사용하여 위젯 기능을 구현합니다:

  • 목적: iOS 위젯 및 Apple Watch 컴플리케이션과의 블록체인 데이터 공유
  • 저장소: 데이터는 장치의 공유 UserDefaults 컨테이너에 저장됩니다
  • 클라우드 동기화 없음: 데이터는 클라우드 서비스나 제3자 서버에 업로드되지 않습니다

5. 워치 앱 연결

Apple Watch 앱을 사용하는 경우 Apple의 WatchConnectivity 프레임워크를 통해 다음 데이터 공유가 수행됩니다:

  • iPhone → Watch: 비트코인 가격, 블록 높이, 수수료, 테마 설정 및 현지화된 문자열
  • 저장소: 모든 데이터는 워치 장치에 로컬로 저장됩니다
  • 개인정보: 외부 서버로 데이터가 전송되지 않습니다. 모든 통신은 로컬 장치에 유지됩니다.

6. 권한

본 앱은 작동하기 위해 특별한 시스템 권한(카메라, 위치, 마이크, 연락처, 사진 등)이 필요하지 않습니다. 앱은 다음만 사용합니다:

  • 인터넷 액세스: 공개 블록체인 데이터 가져오기(익명 요청)
  • 로컬 알림: 선택 사항(위젯 타임라인 업데이트를 활성화한 경우에만)

7. 어린이 개인정보

본 앱은 모든 연령의 사용자에게 적합합니다. 개인정보를 수집하지 않으므로 어린이 개인정보의 수집이나 처리가 없습니다.

8. 데이터 출처

본 앱에 표시된 비트코인 데이터는 다음에서 가져옵니다:

  • Mempool.space: 오픈 소스 비트코인 블록체인 탐색기(https://mempool.space)
  • Binance: 가격 데이터용 암호화폐 거래소 API

당사는 이러한 제3자 출처의 데이터 정확성을 보증하거나 보증하지 않습니다.

9. 정책 변경

개인정보 우선 설계로 인해 중대한 변경 가능성은 낮지만, 본 개인정보 처리방침을 수시로 업데이트할 수 있습니다. 변경 사항은 이 페이지에 업데이트된 유효 날짜와 함께 게시됩니다.

10. 귀하의 권리

데이터를 수집하지 않으므로 액세스, 수정 또는 삭제할 개인 데이터가 없습니다. 장치에 저장된 데이터는 다음과 같이 지울 수 있습니다:

  • 앱 제거(모든 로컬 데이터 지우기)
  • 앱 내에서 설정 → 설정 재설정
