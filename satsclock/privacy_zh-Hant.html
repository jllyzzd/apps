隱私政策

生效日期：2025年12月25日

歡迎使用 SatsClock（"本應用"）。我們重視您的隱私，並致力於保護您的個人資訊。本隱私政策說明了我們關於您資訊的使用實踐。

1. 我們不收集任何資料

SatsClock 是一款隱私優先的比特幣儀表板應用。我們不收集、儲存、傳輸或共享您的任何個人資訊、使用資料或設備資訊。

  • 無需帳號： 您無需註冊或登入即可使用應用的任何功能。
  • 無追蹤器： 我們不使用任何第三方分析 SDK 或使用者行為追蹤服務。
  • 無廣告： 本應用不含廣告，也不展示任何第三方網路廣告。
  • 本地處理： 所有資料顯示和計算均在您的設備本地執行。

2. 網路請求

本應用從以下第三方 API 獲取公共比特幣區塊鏈資料：

  • Mempool.space API： 即時區塊鏈資料（區塊高度、手續費、減半進度）
  • Binance API： 當前比特幣市場價格

重要提示： 這些 API 請求是匿名的。不會傳輸任何個人識別符、設備資訊或使用模式。應用僅讀取公共區塊鏈資料。

3. 資料儲存

本應用在您的設備上儲存最少的使用者偏好資料，僅用於提升使用者體驗：

  • 主題選擇： 您偏好的顏色主題和效果設定
  • 語言偏好： 選定的介面語言（支援8種語言）
  • 顯示設定： 重新整理間隔、保持螢幕常亮、語錄顯示設定
  • 快取資料： 臨時快取的區塊鏈資料以減少網路請求

所有資料透過 iOS UserDefaults 本地儲存，絕不傳輸到任何伺服器。

4. App Groups 和資料共享

本應用使用 iOS App Groups 實現小工具功能：

  • 用途： 與 iOS 小工具和 Apple Watch 複雜功能共享區塊鏈資料
  • 儲存： 資料儲存在您設備的共享 UserDefaults 容器中
  • 無雲端同步： 資料絕不會上傳到任何雲端服務或第三方伺服器

5. 手錶應用連線

如果您使用 Apple Watch 應用，將透過 Apple 的 WatchConnectivity 框架進行以下資料共享：

  • iPhone → Watch： 比特幣價格、區塊高度、費率、主題設定和本地化字串
  • 儲存： 所有資料儲存在您的 Watch 設備本地
  • 隱私： 不會向任何外部伺服器傳輸資料。所有通訊均保留在您的本地設備上。

6. 權限

本應用不需要任何特殊系統權限（如相機、位置、麥克風、通訊錄、照片等）即可執行。本應用僅使用：

  • 網際網路存取： 獲取公共區塊鏈資料（匿名請求）
  • 本地通知： 可選（僅當您啟用小工具時間軸更新時）

7. 兒童隱私

本應用適合所有年齡層使用者。由於我們不收集任何個人資訊，因此不會收集或處理兒童隱私資料。

8. 資料來源

本應用顯示的比特幣資料來源於：

  • Mempool.space： 開源比特幣區塊鏈瀏覽器（https://mempool.space）
  • Binance： 用於價格資料的加密貨幣交易所 API

我們不認可或保證這些第三方來源資料的準確性。

9. 本政策變更

儘管考慮到我們的隱私優先設計，重大變更的可能性不大，但我們可能會不時更新本隱私政策。任何變更將在此頁面發布並更新生效日期。

10. 您的權利

由於我們不收集資料，因此沒有個人資料供您請求存取、修改或刪除。儲存在您設備上的所有資料可透過以下方式清除：

  • 解除安裝應用（清除所有本地資料）
  • 在應用內進入設定 → 重設設定
